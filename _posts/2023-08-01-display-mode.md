[caniuse](https://caniuse.com/?search=display-mode) · [mdn](https://developer.mozilla.org/en-US/search?q=display-mode media query) · [spec](https://drafts.csswg.org/mediaqueries-5/#display-modes)

The `display-mode` CSS media query sets styles based on whether the web page is in an ordinary `browser` tab mode or another mode, such as `fullscreen`, `standalone`, or `minimal-ui`.
